Curso Profesional de Git y Github

Git es un sitema de control de versiones, para trabajar solitario o colaborativo.

Permite trabajar de manera mas ordenada y segura los cambios de un proyecto.

    Los Comandos mas comunes de git son:
  • git init
    • Para iniciar el sistema de versiones de manera local
  • git status
    • Se usa para verificar cuales han sido los cambios realizados y que se mantienen el area de staging o que no han sido trakeados
  • git add
    • si colocamos un espacio y un punto "." (de esta manera "git add .") agregaremos todos los cambios o los archivos nuevos, para luego ser comiteados
  • git commit
    • Si bien se puede usar de esta manera, no se considera una buena practica, lo correcto es usarlo con el flag "-m" para agregarle un comentario o mensaje descriptivo de los cambios realizados
